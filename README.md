# LibraryApp
Η παρούσα εργασία εστιάζει στον σχεδιασμό και την υλοποίηση μιας προηγμένης εφαρμογής Android για τη διαχείριση βιβλιοθηκών, με την ονομασία "LibraryApp". Η εφαρμογή δεν αποτελεί απλώς έναν ψηφιακό κατάλογο, αλλά ένα ολοκληρωμένο οικοσύστημα διαχείρισης πόρων που γεφυρώνει το χάσμα μεταξύ τοπικής αποθήκευσης και υπολογιστικού νέφους (Cloud).
